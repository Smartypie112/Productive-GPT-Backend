# 🧠 Productive-GPT Backend

The backend for **Productive-GPT**, a smart productivity chatbot that helps users manage tasks, improve focus, and build better habits. This API handles task management, chat interactions, and integrates with AI services.

---

## 🚀 Features

- ✅ REST API for task and productivity management  
- 🤖 Chat endpoints for AI-powered productivity guidance  
- 🔐 Environment-based configuration for security  
- 🗄️ Modular folder structure for controllers, routes, services, and models  
- ⚡ Ready for OpenAI API integration or other AI services  

---

## 🛠️ Tech Stack

- Node.js / Express.js  
- dotenv for environment variables  
- cors for cross-origin requests  
- Optional: MongoDB / PostgreSQL for database integration
